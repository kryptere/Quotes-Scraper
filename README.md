
# Quotes Scraper

This project is a web scraper built with Python.

It collects quotes, authors, and tags from https://quotes.toscrape.com/ and saves them into a CSV file.

## Features

- scraping quotes, authors, and tags
- pagination support
- data processing (quote length, tags count)
- CSV export

## Technologies

- Python
- requests
- BeautifulSoup

## How to run

pip install requests beautifulsoup4

python main.py
